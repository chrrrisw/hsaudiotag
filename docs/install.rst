Dependencies
============

``hsaudiotag3k`` requires Python 3.1 and `hsutil <http://hg.hardcoded.net/hsutil>`_ and `py.test <http://codespeak.net/py/dist/>`_.

Installation
============

It's a standard Distribute package, just do::

    python setup.py install

or if you haven't downloaded it yet::

    pip install hsaudiotag
